## 中文说明
本项目整合并优化了适用于 KOReader 的一系列实用用户补丁（User Patches）。最初作为贡献提交于社区讨论区，现建立独立仓库方便机友们下载和维护。

### ✨ 核心功能
* **默认中文显示优化**：彻底解决部分墨水屏设备上原生 KOReader 界面或特定文本默认不优先显示中文的问题。
* **封面切换快捷方式**：新增快捷键或手势触发书籍封面切换，告别繁琐的菜单层级。
* **锁屏与画报优化**：针对墨水屏的锁屏唤醒机制进行微调，完美支持锁屏壁纸及画报的快捷显示。

### 📥 使用方法
1. 下载本仓库中的补丁脚本文件。
2. 将其复制到你设备中 KOReader 的补丁目录下（通常为 `koreader/patches/` 或根据具体补丁说明放置）。
3. 重启 KOReader 即可生效。

## English Description
This repository contains a curated collection of custom user patches for KOReader, originally shared and discussed in the community. 

### ✨ Features
* **Default Chinese Display**: Ensures proper localized font and layout rendering prioritized for Chinese readers.
* **Cover Toggle Shortcut**: Adds an intuitive gesture or shortcut action to toggle book covers instantly.
* **Lockscreen Optimizations**: Refines the sleep/lockscreen behavior tailored for E-ink refresh rates and standby screens.
